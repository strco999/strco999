## ✨ Features (FrontEnd-first)
✅ **UI/UX**
- 🎛️ UI retro consistente, responsive real, componentes reutilizables
- 🧼 Estados claros: loading/empty/error, toasts y feedback de acciones

⚡ **Performance**
- 🖼️ Optimización de imágenes/asset loading
- 🧠 Render eficiente + separación de responsabilidades (UI vs data)

🛒 **E-commerce Flow**
- 🧾 Carrito persistente + resumen de compra
- 💳 Checkout con **Stripe** + estados de orden/pago
- 🏷️ Cupones/descuentos reflejados en totales y detalle

🧰 **Admin Experience**
- 🧑‍💼 Panel Admin (productos, usuarios, cupones, órdenes)
- 🔄 Acciones rápidas y UI pensada para gestión

## 🧭 Qué aporto como FrontEnd
- Diseño de componentes y layouts escalables
- Integración con APIs sin romper UX (manejo de errores y estados)
- Tailwind bien organizado + UI consistente
- Atención al detalle (pixel/performance/flujo)
